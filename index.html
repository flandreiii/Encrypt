capabilities

What Encrypt Does

01 🔐

File Encryption

Encrypt any file — documents, images, archives — into an unreadable ciphertext that only your key can unlock.

02 📝

Text Encryption

Encrypt plain text strings and messages on the fly. Copy the output and share it anywhere — only the key holder can read it.

03 🔓

Decryption

Restore encrypted files and text back to their original form instantly, as long as you have the correct key.

04 🛡️

Strong Algorithms

Powered by proven cryptographic standards. No homebrew crypto, no weak ciphers — just solid, battle-tested security.

05 ⚡

CLI Interface

Simple, scriptable command-line interface. Pipe it, automate it, integrate it into your workflow without friction.

06 🌑

100% Offline

Zero network calls. Ever. Your plaintext and keys never leave your machine — guaranteed by design, not policy.

how it works

Three Steps to Lock Anything

01

Provide your input

Pass a file path or text string as the target. Encrypt handles both with the same simple interface — no mode switching needed.

02

Set your key

Supply a passphrase or let Encrypt generate a cryptographically secure random key. The key is yours — store it safely.

03

Encrypt or decrypt

Run the command. Output is written instantly. Reverse the process any time using the same key — no expiry, no limits.

get started

Install & Run

Clone the repo and you're done. Encrypt is built to be dependency-light so setup takes seconds, not minutes.

Runs anywhere Python runs — Linux, macOS, Windows. No GUI required, no install wizard, no admin rights.

# Clone the repository
$ git clone https://github.com/flandreiii/Encrypt.git

# Enter the directory
$ cd Encrypt

# Install dependencies
$ pip install -r requirements.txt

# Encrypt a file
$ python encrypt.py --encrypt secret.txt

# Decrypt a file
$ python encrypt.py --decrypt secret.txt.enc

> Done. Output written to ./secret.txt.enc
